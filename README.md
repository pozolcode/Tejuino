# Tejuino
CSS Framework focusing on basic elements:

#### Provides:
<ul>
  <li>Flex Layout</li>
  <li>General page Style</li>
  <li>Typography Config</li>
  <li>Navbar lists</li>
  <li>Buttons Styles:
    <ul>
      <li>Default</li>
      <li>Main</li>
      <li>Secondary</li>
    </ul>
  </li>
  <li>Text Styles</li>
  <li>Background and Text Colours</li>
</ul>
